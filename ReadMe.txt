
***Important: Python 2 must be used to run this code.
Running the code:
1. Run the python file "run.py" on command prompt or pycharm.
2. After running the python file "run.py" click or go to the link provided, or you can also just type "localhost:5000" on your browser.
3. You will then be redirected to the web page where the 8-Tile Puzzle Solver is located.

Instructions to play:
1. Click the "Solve" button to shuffle the tiles, and then the puzzle will automatically start solving itself.
2. Click the "Again" button to reload the page, and then you can click the "Solve" button again to reshuffle the tiles in a different order and the puzzle will then automatically start solving itself.
